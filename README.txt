## Overview

WriteWise is an AI-powered writing assistant designed to help you generate engaging and well-crafted blog posts effortlessly. Using advanced generative models, WriteWise allows you to create high-quality blog content based on your specified title, keywords, and desired length.

********************************************************************************************************************************

## Features

- Generate blog posts with a given title and keywords.
- Customize the length of the blog post.
- Ensure the content is original, informative, and engaging.

********************************************************************************************************************************

## Technologies Used

- **Streamlit**: For creating the web interface.
- **Google Generative AI**: For generating blog content based on user input.

********************************************************************************************************************************

## Installation

To set up WriteWise on your local machine, follow these steps:

1. **Clone the Repository**

2. **Create and Activate a Virtual Environment**
python -m venv venv
source venv/bin/activate  


3. **Install the Required Packages**
Make sure you have the requirements.txt file in your project directory. Install the dependencies using:
pip install -r requirements.txt
Set Up API Keys


4. Obtain your Google Generative AI API key and set it in the code or environment variables.


5. **Run the Application**
streamlit run app.py
Replace app.py with the name of your Streamlit script if it's different.
